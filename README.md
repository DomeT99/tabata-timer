# Tabata Timer

## 🏋️ About the App

**Tabata Timer** is an Android application designed to help users efficiently track and manage high-intensity interval training (HIIT) workouts, specifically following the Tabata protocol. The Tabata method is a scientifically-proven training technique consisting of 20-second work intervals followed by 10-second rest periods, repeated over multiple rounds.

This app provides an intuitive and user-friendly interface to guide users through their Tabata sessions, keeping them focused on their fitness goals without worrying about timing.

## 📋 Features

- **Customizable Workouts**: Set the number of rounds and work/rest intervals
- **Real-time Countdown**: Clear, easy-to-read timer display
- **Audio & Visual Alerts**: Get notified when intervals change
- **Progress Tracking**: Monitor your training sessions

## 🚀 Getting Started

### Prerequisites

- [Android Studio](https://developer.android.com/studio) (latest version recommended)
- An Android device or emulator running Android 8.0 or higher
- A [Gemini API Key](https://ai.google.dev/)

### Installation & Setup

1. **Clone/Open the Project**
   - Open Android Studio
   - Select **Open** and choose this project directory
   - Allow Android Studio to fix any incompatibilities during import

2. **Configure Gemini API Key**
   - Create a `.env` file in the project root directory
   - Add your Gemini API key: `GEMINI_API_KEY=your_api_key_here`
   - (See `.env.example` for reference)

3. **Update Gradle Configuration**
   - Remove or comment out this line in `app/build.gradle.kts`:
     ```gradle
     signingConfig = signingConfigs.getByName("debugConfig")
     ```

4. **Run the App**
   - Connect an Android device or start an emulator
   - Click **Run** in Android Studio or press `Shift + F10`
   - The app will build and launch on your device
